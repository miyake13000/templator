テンプレート記法

変数の定義:

{\#- @variable 変数名: type=型, label="ラベル", required=true -\#}

対応している型:

  • string - テキスト入力
  • integer - 整数入力
  • number - 数値入力（小数対応）
  • datetime - 日付+時刻入力
  • boolean - チェックボックス
  • select - プルダウン（options指定必須）
  • array - カンマ or 改行区切りのリスト

追加オプション:

  • label - フォームに表示するラベル
  • required - 必須項目（true/false）
  • options - 選択肢の配列 [例: options=["選択1", "選択2"]]
  • min, max - 数値の最小・最大値
  • default - デフォルト値
  • placeholder - プレースホルダー

Jinja2構文:

  • 変数: {{ "{{ variable }}" }}
  • 繰り返し: {{ "{% for item in items %}" }}...{{ "{% endfor %}" }}
  • 制御構文には '-' を付けることで制御構文による空行を削除できます。
